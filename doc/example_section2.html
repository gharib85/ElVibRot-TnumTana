This namelist has numerous parameters, but most of them were used for the old coordinate transformation procedure (see . section transformations) They are kept for the compatibility with previous input data files.

nrho

nrho enables one to select the volume element:


0: Euclidian (rho=jacobian)

To be use only for checking purpose.

1: Wilson (rho=1)

2: product of 1D-volume element: dR.dphi.sin(th)dth…

10: same as 1, without extrapotential term

20: same as 2, but without extrapotential term

2

WriteT

When the value is set to T, the subroutines write the KEO (f2, f1 or GG, g …).

To be use only for checking purpose.
